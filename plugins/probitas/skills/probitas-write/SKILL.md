## Use Agent

For scenario creation/editing, use the `scenario-writer` agent:

> Task agent: scenario-writer

The agent has templates, checklists, and API reference links.

## Commands

- `/probitas:new` - Generate scenario template
- `/probitas:check` - Format, lint, type check
